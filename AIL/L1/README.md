# Lesson 01 HTML PPT

This folder contains the HTML slide deck for Lesson 01.

## Files

- `index.html`: the full slide deck
- `style.css`: deck-specific styling
- `assets/`: local runtime, theme, font, and animation files copied from the `html-ppt` skill so the deck runs inside this repo

## Usage

Open `index.html` in a browser.

## Keyboard

- `竊秦 `竊蛋: navigate
- `F`: fullscreen
- `O`: overview
- `S`: presenter mode
- `T`: cycle light themes

## Theme Default

The deck starts in `corporate-clean` and can cycle through:

- `corporate-clean`
- `academic-paper`
- `minimal-white`
